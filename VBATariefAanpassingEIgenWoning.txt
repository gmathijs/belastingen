Public Function TariefAanpassingEIgenWoning2025(Aftrek As Double, Inkomen As Double) As Double
' Hypotheekrenteaftrek 2020 tegen 49%, in 2024 omlaag naar ongeveer 37%
' Wie een hoger inkomen heeft en een inkomen heeft in een belastingschijf met een hoger belastingtarief,
' kan meer van de betaalde hypotheekrente terugkrijgen van de fiscus. Maar dat voordeel neemt steeds verder af als het aan het kabinet Rutte 3 ligt.
' De teruggave kon oplopen tot 52% van de betaalde hypotheekrente in 2012, maar dat is al lang niet meer zo:
' In 2019 ligt het maximum op 49% bij een inkomens grens van schijf 3 68.507
' In 2020 ligt het maximum op 46% bij een inkomens grens van schijf 3 68507 @ 49.5%
' in 2023 ligt het maximum op 49.5-12.57 = 36.93% bij een inkomens grens hoger dan  schijf 3 73031 @ 49.5%
' in 2024 ligt het maximum op 49.5-12.53 = 36.97% bij een inkomens grens hoger dan  schijf 3 75518 @ 49.5%

  
  If Inkomen > 75518 Then
    TariefAanpassingEIgenWoning2025 = Aftrek * 0.1253 ' van 49.5 naar 36.97%
  Else
    TariefAanpassingEIgenWoning2025 = 0#
  End If
    

End Function
